# Template de operação

Copie esta pasta para `arquivos/sua-operacao/` e depois copie também:

```text
arquivos/operacao-albania/index.html
arquivos/operacao-albania/css/
arquivos/operacao-albania/js/
```

O visualizador lê `data/operation.js`, então cada operação pode ter seus próprios dados sem alterar o motor principal.
